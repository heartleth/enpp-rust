operators

English++ has various operators.

  • 산술 연산자: + - * / % plus minus

    • +, plus : 덧셈 연산자
    • -, minus: 뺼셈 연산자
    • * : 곱셈 연산자
    • / : 나눗셈 연산자
    • % : 나머지 연산자

    The plus, minus, -, and % operators must be separated. It should be written as 1 - 1, not 1-1.

    You can use 1+1.

  • Binary operators: or, and, <<,>>, <,>, is, is not

    • or, and
    • <<,>>: Shift operator
    • <,>: Compare operator
    • is: Equivalent to '=='.

  • if then else 연산자

    if 1 is 1 then 2 else 3의 값은 2입니다.

    if 1 is 2 then 2 else 3의 값은 3입니다.

    if A then B else C의 값은 A가 참이면 B이고, 거짓이면 C입니다.

    if A then B else if C then D else E처럼 여러 조건을 걸 수도 있습니다.

    You can also change a line. 

    if A then B
else if C then D
else E

  • 'as' and function complex operator, function operator, unary function operator

    a as 10 Assign 10 for a. The value of the expression is 10.

    a func= 10 Assign f(a, 10) into a. The value of the expression is the value assigned into a.

    a func! 10 The value of the expression is f(a, 10).

    func: a The value of the expression is f(a).

Of course, you can concatenate operators in the format 1 + 1 + 1 + 1.

And of course, you can use parentheses in the format 1 + (1 + 1) + 1. It is recommended to use parentheses for functional compound operators and (unary) function operators.

For

English++ supports arrays that can be simply written as [1..10]. [1..10] is an array containing numbers from 1 to 10 in order with a difference of 1. If you want an array up to 20, you can write [1..20]. If you want the difference of 2, just write [1...20] and 3 dots in the middle. Those numbers can only be constants. [a..b] is not possible.

Using that array, we can write code that prints numbers from 1 to 10. We haven't learned types yet, but let's move on.

For i of [1..10]
    it println about i
